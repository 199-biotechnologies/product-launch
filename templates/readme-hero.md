## Required badge set (shields.io)

| Badge | What it signals |
|-------|-----------------|
| stars | Social proof |
| license | Legal trust |
| downloads / installs | Adoption (only include if numbers are good) |
| CI status | Maintained (only if actually green) |
| version | Current release |
| twitter follow | Creator credibility |

Skip vanity badges (last-commit, etc.). They create noise.
